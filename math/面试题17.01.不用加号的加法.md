#### 面试题 17.01. 不用加号的加法
设计一个函数把两个数字相加。不得使用 + 或者其他算术运算符。

示例:
```
输入: a = 1, b = 1
输出: 2
```
提示：
- a, b 均可能是负数或 0
- 结果不会溢出 32 位整数

#### 解法
```go
func add(a int, b int) int {
    for b != 0 {
        sum := a^b // 不进位部分：3 ^ 5 = 11 ^ 101 = 110=6
        jinwei := (a&b)<<1 // 需要进位的部分，再前移1位: 3&5= 1, 左移是进位的一位=10=2
        a = sum
        b = jinwei
    }
    return a
}
```
